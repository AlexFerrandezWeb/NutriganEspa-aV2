2. Datos que Recopilamos

Recopilamos los siguientes tipos de datos personales:

  • Datos de identificación: Nombre, apellidos, DNI/NIE
  • Datos de contacto: Dirección postal, teléfono, email
  • Datos de navegación: Dirección IP, cookies, datos de uso del sitio web
  • Datos comerciales: Historial de compras, preferencias de productos

3. Finalidad del Tratamiento

Los datos personales se tratan con las siguientes finalidades:

  • Gestión comercial: Procesamiento de pedidos, facturación, envío de productos
  • Atención al cliente: Resolución de consultas y reclamaciones
  • Marketing: Envío de comunicaciones comerciales (con consentimiento)
  • Mejora del servicio: Análisis de uso del sitio web para mejorar la experiencia del usuario
  • Cumplimiento legal: Cumplimiento de obligaciones fiscales y contables

4. Base Legal del Tratamiento

El tratamiento de datos se basa en:

  • Ejecución de contrato: Para la gestión de pedidos y servicios contratados
  • Consentimiento: Para el envío de comunicaciones comerciales
  • Interés legítimo: Para la mejora del servicio y análisis de uso
  • Cumplimiento legal: Para el cumplimiento de obligaciones fiscales

5. Conservación de Datos

Los datos personales se conservarán durante:

  • Datos comerciales: 6 años desde la última transacción (obligación fiscal)
  • Datos de marketing: Hasta que se retire el consentimiento
  • Datos de navegación: Máximo 2 años
  • Datos de contacto: Hasta que se solicite la supresión

6. Compartir Datos

Sus datos personales pueden ser compartidos con:

  • Proveedores de servicios: Empresas de transporte, procesadores de pago
  • Autoridades competentes: Cuando sea requerido por ley
  • Partners comerciales: Solo con su consentimiento expreso

No vendemos ni alquilamos sus datos personales a terceros.

7. Sus Derechos

Como titular de los datos, tiene derecho a:

  • Acceso: Conocer qué datos tenemos sobre usted
  • Rectificación: Corregir datos inexactos o incompletos
  • Supresión: Solicitar la eliminación de sus datos
  • Limitación: Restringir el tratamiento de sus datos
  • Portabilidad: Recibir sus datos en formato estructurado
  • Oposición: Oponerse al tratamiento de sus datos

Para ejercer estos derechos, puede contactar con nosotros en: javiernutrigan@gmail.com

8. Seguridad de los Datos

Implementamos medidas técnicas y organizativas apropiadas para proteger sus datos personales contra:

  • Acceso no autorizado
  • Alteración, divulgación o destrucción no autorizada
  • Pérdida accidental

Utilizamos cifrado SSL, firewalls y controles de acceso para garantizar la seguridad de sus datos.

9. Cookies

Nuestro sitio web utiliza cookies para mejorar la experiencia del usuario. Para más información, consulte nuestra Política de Cookies.

10. Menores de Edad

Nuestros servicios no están dirigidos a menores de 16 años. No recopilamos intencionalmente datos personales de menores sin el consentimiento de sus padres o tutores legales.

11. Cambios en la Política

Nos reservamos el derecho de modificar esta política de privacidad. Los cambios serán notificados a través del sitio web con al menos 30 días de antelación.
